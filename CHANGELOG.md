# 更新日志

## 0.1.15（2026-07-18）— 设计与体验大版本

**全新登录页**
- 沉浸式设计：WebGL 云雾动态背景（品牌色光斑漂移）+ 墨面一体表单
- 三段入场动效、失败摇晃反馈、成功淡出过场
- 低性能/无 WebGL 设备自动降级静态背景；跟随系统深浅色

**品牌**
- 应用内品牌落地「Ark Photo」：侧边栏字标、页面标题、favicon

**全应用动效打磨**
- Toast 通知进出场对称动画
- 扫描进度条出现/消失不再挤动照片墙（FLIP 化，零重排）
- 拖拽上传遮罩重设计（整页玻璃化 + 细虚线投放框）
- 弹窗统一入场/退场动效；新建相册弹窗重设计
- 收藏心形按压反馈
- 全部动效尊重系统「减弱动态效果」设置

**交互修复**
- OCR 识别文字改为折叠入口 + 浮层展示，不再把详情面板撑变形
- Safari 拖拽图片不再误触上传弹层/多选
- Shift 区间连选（支持 Cmd+Shift 叠加）、右键不再误触多选
- 批量选择性能优化（工具栏增量更新，复选框不再闪屏）
- 时间线首屏偶发空白修复；搜索进详情不再显示「未知拍摄时间」

**部署**
- 新增 `THUMB_WORKERS` 环境变量：低内存 NAS（512MB–1GB）设 `1` 可将缩略图生成改为串行，内存峰值降到单张解码水平

> 本版本先发布 Docker 镜像（`ghcr.io/siriusswh/nas-photo:0.1.15`）；飞牛 .fpk 随后跟进。

## 0.1.14（2026-07-15）— 首个公开发布版本

**飞牛原生应用（.fpk）**
- 官方 fnpack 打包，应用中心手动安装，桌面图标直达 Web 界面
- 「授权文件夹」接入飞牛权限体系，授权即可作为图库使用
- 以专用非 root 用户运行；升级保留全部数据

**核心功能**
- 时间线（月份分组 / 6 档缩放 / 排序筛选 / 多选批量）
- 相册、收藏、回收站（软删可恢复）
- Live Photo 自动配对播放；视频流式播放
- RAW 21 种格式 + RAW/JPEG 自动配对 + exiftool 完整 EXIF
- 详情页：EXIF / GPS 地图 / 胶片条
- 截图自动识别、拍摄时间多级解析（EXIF → 文件名 → mtime）

**AI（可选，一键开启）**
- 人脸识别 + 聚类，按人物浏览
- 中文/英文自然语言搜图（CLIP）
- 照片文字识别与搜索（OCR，中英多语）
- 框架与模型按需下载到数据盘（约 3.4GB，国内自动镜像源），重建/升级不重下

**修复**
- 相册页与照片页统一全宽布局
- 图库管理文案与交互优化、自定义 tooltip
